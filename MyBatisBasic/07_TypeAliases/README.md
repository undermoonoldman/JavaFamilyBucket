# ***使用 typeAliases 配置别名***

1. ## typeAliases 配置别名,只能配置domain中类的别名

2. ## package 配置别名的包,指定包后,下面的实体类都会注册别名,类名即为别名,不区分大小写

3. ## typeAlias 中的属性

   1. ### type:指定全类名

   2. ### alias:指定别名,指定别名后不区分大小写
